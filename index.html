France's first conversational AI — SNCF

Led the design and delivery of the first large-scale NLP chatbot in France for Voyages-SNCF.com. Integrated with Facebook Messenger using Stanford NLP + OpenNLP, enabling users to search and book train tickets directly in chat. Launched December 2016, resulting in a 50% increase in bookings and about 240M€ in revenue.

Global API gateway architecture — Dior

Managed the development and architected the enterprise GraphQL API gateway that powers Dior's retail across the world. Delivered it worldwide within 4 months while managing concurrent deployments across Europe and China. Enhanced data retrieval performance by 60% by optimizing query resolution and caching strategies.

Zero-downtime OIDC migration — Club Med

Managed the development and maintenance of the public REST API infrastructure for global tourism leader. Unified authentication covering millions of users, employees and customers. A full cross-functional program spanning security, engineering, QA, and stakeholder management across multiple countries.
