I led the development of France's first large-scale conversational AI at SNCF (Facebook Messenger, NLP), architected Dior's global API gateway, and drove a zero-downtime OIDC migration at Club Med covering millions of users.

France's first conversational AI — SNCF

Led the design and delivery of the first large-scale NLP chatbot in France for Voyages-SNCF.com. Integrated with Facebook Messenger using Stanford NLP + OpenNLP, enabling users to search and book train tickets directly in chat. Launched December 2016, resulting in a 50% increase in bookings.

Global API gateway architecture — Dior

Architected the enterprise API gateway that powers Dior's retail across the world. Delivered a full developer enablement program to standardize API governance across all brand properties globally.

Zero-downtime OIDC migration — Club Med

Unified authentication across four global platforms with zero service interruption, covering millins of users. A full cross-functional program spanning security, engineering, QA, and stakeholder management across multiple countries.
